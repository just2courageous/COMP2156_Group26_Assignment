COMP2156 is a DevOps course offered by GBC it prepares students to work in real workflows by assigning work to be done as a group. 
Furthermore students will learn the procedures, protocols and best practices that a DevOps engineer brings to a workplace environment 
to increase productivity. Analyze and evaluate various developer operation implementation available and implement hands-on work using 
industry standard toolkit. Alongside that students will gain knowledge on software development cycle which is a critical component.
